## NodeJs + Mongodb + Express + Socket.io

**a simple blog system powered by Nodejs, Mongoldb, Express and Socket.io**

### How to use?

> npm install

open the dbconnect.js file. config your database name and password.

### How to start it?

> nodemon server.js
